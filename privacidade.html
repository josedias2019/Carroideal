Política de Privacidade

Última actualização: Março 2026

Comprometemo-nos a proteger os teus dados. Nunca os vendemos, nunca os partilhamos com terceiros para fins comerciais.

1. Quem somos

O CarroIdeal.pt é um serviço de consultoria automóvel independente, gerido por José Dias, com sede em Vila Verde, Braga, Portugal.

Contacto: rooms4you2024@gmail.com · +351 911 954 717

2. Que dados recolhemos

Recolhemos apenas os dados que tu nos forneces voluntariamente:

  • Nome — para te identificarmos quando te contactamos
  • Número de telefone — para contacto comercial sobre o teu carro ideal
  • Respostas ao questionário — anónimas, usadas apenas para calcular o teu perfil psicológico
  • Distrito — obtido via GPS com a tua autorização, para mostrar stands próximos

Não recolhemos dados de pagamento, documentos de identificação, nem dados sensíveis.

3. Para que usamos os dados

  • Contactar-te para discutir as recomendações de carros
  • Encaminhar o teu contacto para stands parceiros verificados (quando aplicável)
  • Melhorar o serviço e as recomendações

Nunca usamos os teus dados para publicidade de terceiros, marketing não solicitado ou qualquer fim diferente do acima descrito.

4. Com quem partilhamos os dados

Os teus dados podem ser partilhados com stands parceiros verificados do CarroIdeal.pt, exclusivamente para fins de contacto comercial relacionado com a tua pesquisa de carro.

Todos os stands parceiros são verificados por nós antes de aderirem ao programa e comprometem-se a não usar os dados para outros fins.

Não vendemos nem partilhamos dados com terceiros fora da rede CarroIdeal.pt.

5. Quanto tempo guardamos os dados

Os teus dados são guardados durante 12 meses após o último contacto. Após esse período são eliminados automaticamente.

Podes pedir a eliminação dos teus dados a qualquer momento — ver secção 7.

6. Onde guardamos os dados

Os dados são guardados em:

  • Google Sheets — servidores da Google na União Europeia
  • Gmail — para notificações internas

Todos os serviços usados cumprem o RGPD.

7. Os teus direitos

Ao abrigo do RGPD, tens direito a:

  • Acesso — saber que dados temos sobre ti
  • Rectificação — corrigir dados incorrectos
  • Eliminação — pedir que apaguemos os teus dados
  • Oposição — opor-te ao tratamento dos teus dados
  • Portabilidade — receber os teus dados em formato legível

Para exercer qualquer destes direitos, contacta-nos por email ou telefone. Respondemos em até 72 horas.

8. Cookies

O CarroIdeal.pt não usa cookies de rastreamento, publicidade ou análise de terceiros. O site funciona sem guardar informação no teu dispositivo.

9. Alterações a esta política

Podemos actualizar esta política ocasionalmente. A data de actualização está sempre visível no topo desta página. Alterações significativas serão comunicadas por email quando aplicável.

10. Contacto e reclamações

Para qualquer questão sobre privacidade:

  • Email: rooms4you2024@gmail.com
  • Telefone: +351 911 954 717

Se não ficares satisfeito com a nossa resposta, podes apresentar reclamação à CNPD (Comissão Nacional de Protecção de Dados) em cnpd.pt.
